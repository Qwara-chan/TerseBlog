# Hello World

私の **Astro ブログ** へようこそ！このサイトの最初の投稿です。

有名な方程式：$E = mc^2$

## はじめに

このブログは、モダンな静的サイトジェネレーターである Astro で構築されています。はじめにどうぞ。

### 前提条件

- Node.js 18 以降
- コードエディタ（VS Code 推奨）
- HTML と Markdown の基礎知識

### インストール

新しい Astro プロジェクトを作成します：

```bash
npm create astro@latest
cd my-blog
npm run dev
```

## 機能

- **標準で高速** — JS ゼロの純粋な静的 HTML
- **Markdown 対応** — frontmatter 付きで Markdown で執筆
- **KaTeX 数式** — LaTeX 方程式をレンダリング（例：$\int_a^b f(x)\,dx$）
- **検索** — Fuse.js による全文検索
- **ダークモード** — システム設定に従います

### 数式の例

インライン数式：$a^2 + b^2 = c^2$

ディスプレイ数式：

$$
\sum_{k=1}^{n} k = \frac{n(n+1)}{2}
$$
