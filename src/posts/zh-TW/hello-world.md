# 你好世界

歡迎來到我的 **Astro 部落格**！這是本網站的第一篇文章。

著名的方程式：$E = mc^2$

## 入門指南

這個部落格是使用 Astro 這個現代靜態網站產生器所打造，以下是入門方式。

### 前置需求

- Node.js 18 或更新版本
- 一個程式碼編輯器（推薦 VS Code）
- HTML 與 Markdown 的基礎知識

### 安裝方式

建立一個新的 Astro 專案：

```bash
npm create astro@latest
cd my-blog
npm run dev
```

## 功能特色

- **預設極速** — 純靜態 HTML，零 JS 負擔
- **Markdown 支援** — 使用 Markdown 撰寫內容並搭配 frontmatter
- **KaTeX 數學** — 渲染 LaTeX 方程式，例如 $\int_a^b f(x)\,dx$
- **搜尋** — 使用 Fuse.js 提供全文搜尋
- **深色模式** — 自動依循系統偏好

### 數學範例

行內數學：$a^2 + b^2 = c^2$

展示數學：

$$
\sum_{k=1}^{n} k = \frac{n(n+1)}{2}
$$
