# 你好世界

欢迎来到我的 **Astro 博客**！这是本站的第一篇文章。

著名的方程式：$E = mc^2$

## 入门指南

这个博客是使用 Astro 这个现代静态网站生成器所打造，以下是入门方式。

### 前置需求

- Node.js 18 或更新版本
- 一个代码编辑器（推荐 VS Code）
- HTML 与 Markdown 的基础知识

### 安装方式

创建一个新的 Astro 项目：

```bash
npm create astro@latest
cd my-blog
npm run dev
```

## 功能特性

- **默认极速** — 纯静态 HTML，零 JS 负担
- **Markdown 支持** — 使用 Markdown 编写内容并搭配 frontmatter
- **KaTeX 数学** — 渲染 LaTeX 公式，例如 $\int_a^b f(x)\,dx$
- **搜索** — 使用 Fuse.js 提供全文搜索
- **深色模式** — 自动遵循系统偏好

### 数学示例

行内数学：$a^2 + b^2 = c^2$

展示数学：

$$
\sum_{k=1}^{n} k = \frac{n(n+1)}{2}
$$
